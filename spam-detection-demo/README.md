# Spam Detection Demo

This project is a simple spam detection demo that allows users to input text and receive feedback on whether the text is likely to be spam or not. The application is built using HTML, CSS, and JavaScript, and it runs on a Node.js server.

## Project Structure

```
spam-detection-demo
├── src
│   ├── index.html       # Main HTML document
│   ├── styles.css       # Styles for the application
│   └── script.js        # JavaScript for spam detection logic
├── server.js            # Node.js server to serve the application
├── package.json         # npm configuration file
└── README.md            # Project documentation
```

## Features

- User-friendly interface for inputting text.
- Real-time spam detection feedback.
- Simple and clean design.

## Getting Started

To set up and run the application locally, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd spam-detection-demo
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Start the server:**
   ```
   node server.js
   ```

4. **Open your browser and navigate to:**
   ```
   http://localhost:3000
   ```

## Usage

- Enter text in the input field and click the "Check Spam" button.
- The application will analyze the text and display whether it is likely to be spam.
